# 🔐 Hash Security Analyzer

## 📌 Overview
This is a console-based Hash Security Analyzer that generates and compares cryptographic hashes using MD5, SHA-1, SHA-256, and SHA-512. It demonstrates core cryptography concepts such as hashing, security strength, entropy, and the avalanche effect.

---

## 🚀 Features
- Generate hashes using multiple algorithms (MD5, SHA-1, SHA-256, SHA-512)
- Compare security levels of hashing algorithms
- Demonstrate the avalanche effect (small input change → big output change)
- Analyze hash entropy (randomness measurement)
- Display fingerprint distribution (hex character frequency 0–f)
- Provide security recommendations

---

## 🧠 What You Learn
- Basics of cryptographic hashing
- Why MD5 and SHA-1 are insecure
- How SHA-256 and SHA-512 improve security
- Importance of entropy in cryptography
- How small input changes affect hash outputs
